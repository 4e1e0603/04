## Install

- Clone the repository.
  ```  
  git clone https://github.com/groundf/course-python.git
  ```
- Move to the folder.
  ```
  cd course-python
  ```
- Create a virtual environment, see &hellip;
- Install dependencies.
  ```
  pip install -e .
  ```
- Run the examples.
