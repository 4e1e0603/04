## Installation

- Clone the repository.
  ```
  git clone https://github.com/groundf/course-python.git
  ```
- Move to the folder.
  ```
  cd course-python
  ```
- Install dependencies.
  ```
  py -m venv .venv
  source\Scripts\activate
  pip install .
  ```

And read the course content.
