Prerequisites

We assume the following

  • You have installed winget to easily install all other sofwtare, namely

    • Git: winget install -e --id Git.Git

    • Terminal: winget install -e --id Microsoft.WindowsTerminal

    • Visual Studio Code: winget install -e --id VSCodium.VSCodium

    • Python

  • You know how to use console (terminal) and basic Windows/Linux commands.
    …We will call the Python and Git from command line. If not go here.

  • You know how to write characters such as #, %, ^, &, |, ~, *, +, -, /, (, ), {, }, [, ], _, ', ".
    …We will write a lot! If not go here.
